\doxysubsection{Detailed Description}
This class is responsible for calculating and updating vorticity for a given stream function that is performed by the Solver CG class. 

\begin{DoxyNote}{Note}
This class takes in all the information about ranks including their coordinates and rank numbers to discretise the working domain on the cartesian grid accordingly 
\end{DoxyNote}
